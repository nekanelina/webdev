### Usage

1. Rename the `.envexampl`e to `.env` and add your `MONGO_URI`

2. Install dependencies

```sh
# Backend deps
npm install
```

```sh
# Frontend deps
cd frontend
npm install
```


3. Run Server

```sh
npm run server
```

3. Run FrontEnd

```sh
npm run client
```
